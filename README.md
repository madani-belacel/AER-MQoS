# AER-MQoS-Code

Source code and simulation data for the AER-MQoS routing protocol.

This repository contains the Contiki-NG implementation and the measured Cooja simulation data (N=9 seeds) used in the associated research paper.

## Content

- `code_source_AER_MQoS/` → Firmware implementation
- `simulations/` → Cooja scenarios and execution scripts
- `Section-tex/sim/multi_seed/` → Measured simulation results
- `INSTALL.md` → Instructions to compile and reproduce
- `NAMING.md` → Naming convention note
