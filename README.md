# AER-MQoS

**Source Code and Measured Simulation Data**

This repository contains the Contiki-NG firmware implementation and the measured Cooja simulation data (N=9 independent seeds) for the AER-MQoS routing protocol.

## Content

- `code_source_AER_MQoS/` — Firmware source code
- `simulations/` — Cooja scenarios and execution scripts
- `sim/` — Measured simulation results (N=9 seeds)
- `INSTALL.md` — Setup and reproduction guide
- `NAMING.md` — Naming convention note
