# AER-MQoS

Source code and measured simulation data for the AER-MQoS routing protocol.

This repository contains the Contiki-NG firmware implementation and the measured Cooja data (N=9 seeds) used in the associated research paper.

## Content

- `code_source_AER_MQoS/` → Firmware source code
- `simulations/` → Cooja scenarios and scripts
- `sim/` → Measured simulation data
- `INSTALL.md` → Reproduction instructions
- `NAMING.md` → Naming convention explanation
