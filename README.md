# AER-MQoS-Code

Source code and measured simulation data for the **AER-MQoS** routing protocol.

This repository contains the Contiki-NG firmware implementation and the measured Cooja simulation results (N=9 independent seeds).

## Content

- `code_source_AER_MQoS/` → Firmware source code
- `simulations/` → Cooja scenarios and run scripts
- `sim/` → Measured simulation data (N=9 seeds)
- `INSTALL.md` → Setup and reproduction instructions
- `NAMING.md` → Naming convention explanation
