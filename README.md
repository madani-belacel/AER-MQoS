# AER-MQoS-Code

Source code and simulation data for the AER-MQoS routing protocol.

This repository contains the Contiki-NG implementation and the measured Cooja simulation data used in the associated research paper (currently under review).

## Content

- `code_source_AER_MQoS/` → Firmware implementation
- `simulations/` → Cooja scenarios and execution scripts
- `Section-tex/sim/multi_seed/` → Measured data (N=9 independent seeds)
- `INSTALL.md` → Instructions to compile and reproduce experiments
- `NAMING.md` → Naming convention note

## Note

The full research paper (LaTeX sources and camera-ready PDF) is submitted to a journal and will be made publicly available after acceptance.
