BUILD_NOTES.txt — AER-MQoS campaign provenance template
======================================================
Populate this file before each campaign with:
  - git describe --always   (run from the Contiki-NG checkout)
  - gcc --version           (compiler used for Cooja build)
  - date                    (campaign date)
  - Host machine            (OS, kernel, RAM)
  - Any notes on environment changes

Example:
  contiki-ng: v6.8-123-gabc1234
  gcc: (Ubuntu 11.4.0-1ubuntu1~22.04) 11.4.0
  date: 2026-06-01
  host: x86_64 GNU/Linux, 16 GB RAM
